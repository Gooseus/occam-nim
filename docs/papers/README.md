# Reference Papers

This folder contains key papers on Reconstructability Analysis (RA) for offline reference.

## Papers

### Introductory

| File | Authors | Year | Description |
|------|---------|------|-------------|
| `zwick-2004-overview-of-ra.pdf` | Martin Zwick | 2004 | **An Overview of Reconstructability Analysis** — The definitive introduction to RA methodology, published in *Kybernetes* |
| `zwick-2018-intro-to-ra.pdf` | Martin Zwick | 2018 | **Introduction to Reconstructability Analysis** — Accessible presentation slides covering RA fundamentals |

### Tutorials & Implementation

| File | Authors | Year | Description |
|------|---------|------|-------------|
| `zwick-2020-ra-and-occam.pdf` | Martin Zwick | 2020 | **Reconstructability Analysis and Its OCCAM Implementation** — Tutorial presented at ICCS 2020, demonstrates OCCAM software |

### Applications

| File | Authors | Year | Description |
|------|---------|------|-------------|
| `shervais-2010-gene-gene-interactions.pdf` | Shervais, Kramer, Westaway, Cox, Zwick | 2010 | **RA as a Tool for Identifying Gene-Gene Interactions** — Detecting epistasis in genetic studies |

  title={An overview of reconstructability analysis},
  author={Zwick, Martin},
  journal={Kybernetes},
  volume={33},
  number={5/6},
  pages={877--905},
  year={2004},
  publisher={Emerald Group Publishing Limited}
}
